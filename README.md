### 🤖 03 &nbsp;·&nbsp; Loan Approval Prediction — ML Pipeline

<img align="right" width="90" src="https://img.icons8.com/fluency/96/artificial-intelligence.png"/>

**`Python`** &nbsp;|&nbsp; **`Scikit-learn`** &nbsp;|&nbsp; **`Pandas`** &nbsp;|&nbsp; **`Feature Engineering`** &nbsp;|&nbsp; **`Classification`**

> **Business Problem:** Manual loan screening is slow, inconsistent and prone to human bias. Automate approval decisions using applicant financial and demographic data — improving speed and accuracy.

**🔍 What I Did:**
- Performed deep **EDA** on 600+ loan records — class imbalance detection, feature-target correlations, distribution plots
- Engineered new features: **income-to-loan ratio**, **credit history flag**, **dependents-adjusted income**
- Trained and compared **Logistic Regression, Decision Tree & Random Forest** classifiers with K-Fold cross-validation
- Tuned hyperparameters via **GridSearchCV** and evaluated with Precision, Recall, F1-Score, and ROC-AUC
- Produced a clean **model comparison report** with confusion matrices for all three algorithms

**📈 Key Outcomes:**

| Model | Accuracy | ROC-AUC | Verdict |
|:---|:---:|:---:|:---:|
| Logistic Regression | 80.5% | 0.83 | Baseline ✅ |
| Decision Tree | 78.2% | 0.79 | Overfitting ⚠️ |
| **Random Forest** | **83.1%** | **0.87** | **Best Model 🏆** |
| Majority Baseline | 68.7% | — | Naive ❌ |
